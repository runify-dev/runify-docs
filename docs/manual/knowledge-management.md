# 知识库

## 1 功能说明

知识库模块用于统一管理知识库、文件夹和文档。你可以在这里创建知识库，维护文档目录，并编辑文档内容。

---

## 2 页面说明

知识库页面由两部分组成：

| 区域 | 说明 |
| --- | --- |
| 左侧目录树 | 管理文件夹、知识库和文档目录 |
| 右侧内容区 | 查看知识库列表、文档列表、文档内容和知识库设置 |

### 2.1 列表页

列表页中主要包含：

- 搜索框
- 创建按钮
- 知识库卡片
- 卡片操作菜单

![知识库页面](/img/manual/knowledge/knowledge-home.png)

---

## 3 操作步骤

### 3.1 新建知识库

1. 进入 **知识库** 页面。
2. 在左侧选择目标文件夹。
3. 点击右上角 **创建**。
4. 选择 **创建知识库**。

![创建知识库按钮](/img/manual/knowledge/knowledge-create-button.png)

也可以从左侧目录树中创建：

1. 在左侧找到目标文件夹。
2. 点击节点右侧操作菜单。
3. 选择 **创建 -> 知识库**。

![目录树菜单](/img/manual/knowledge/knowledge-tree-menu.png)

4. 上传知识库图标。
5. 填写知识库名称。
6. 填写知识库描述。
7. 点击 **创建**。

![新建知识库弹窗](/img/manual/knowledge/knowledge-create-dialog.png)

### 3.2 查看和修改知识库

打开知识库后，可以查看文档目录，并在设置页修改知识库名称和描述。

1. 在列表中点击知识库卡片。
2. 或在左侧目录树中点击知识库名称。
3. 点击左侧底部 **设置**。
4. 修改知识库名称和描述。
5. 点击右上角 **保存**。

![知识库设置](/img/manual/knowledge/knowledge-setting.png)

### 3.3 新建文件夹和文档

进入知识库后，可以在当前目录下继续维护文档结构。

1. 打开目标知识库。
2. 进入需要新增内容的目录。
3. 点击右上角 **创建**。
4. 选择 **新建文件夹** 或 **新建文档**。
5. 按提示填写名称并完成创建。

![新建文件夹和文档](/img/manual/knowledge/knowledge-doc-create.png)

### 3.4 编辑文档内容

知识库文档使用编辑器维护内容。

1. 在目录树或内容区点击目标文档。
2. 进入文档编辑页后修改内容（写入的文档内容会自动保存）。
3. 点击右下角 **保存** 按钮保存文档。

![文档编辑](/img/manual/knowledge/knowledge-editor.png)

### 3.5 删除知识库或文档

1. 找到要删除的知识库、文件夹或文档。
2. 点击右侧操作菜单。
3. 选择 **删除**。
4. 在确认框中再次确认。

![删除知识库或文档](/img/manual/knowledge/knowledge-delete.png)
