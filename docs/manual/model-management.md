# 模型

## 1 功能说明

模型模块用于统一管理系统中的模型配置。你可以在这里新建模型、选择供应商和模型类型、填写认证信息，并维护模型参数。

---

## 2 页面说明

模型页面由两部分组成：

| 区域 | 说明 |
| --- | --- |
| 左侧目录树 | 管理文件夹和模型 |
| 右侧内容区 | 查看列表、创建模型、编辑模型配置 |

### 2.1 列表页

列表页中主要包含：

- 搜索框
- 新建模型
- 模型卡片
- 卡片操作菜单
![模型页面](/img/manual/model/model-home.png)


---

## 3 操作步骤

### 3.1 新建模型

1. 进入 **模型** 页面。
2. 在左侧选择目标文件夹。
3. 点击右上角 **新建模型**。

![新建模型按钮](/img/manual/model/model-create-button.png)

也可以从左侧目录树中创建：

1. 在左侧找到目标文件夹。
2. 点击节点右侧操作菜单。
3. 选择 **新建模型** 或 **创建 -> 模型**。

![目录树菜单](/img/manual/model/model-tree-menu.png)

4. 填写名称。
5. 选择供应商。
6. 选择模型类型。
7. 选择或输入模型名称。
8. 填写认证信息。
9. 点击 **创建**。
![新建模型弹窗](/img/manual/model/model-create-dialog.png)

### 3.2 选择供应商和模型类型

选择供应商后，页面会显示可选的模型类型和模型名称。

![模型类型](/img/manual/model/model-type-select.png)

![供应商和认证信息](/img/manual/model/model-provider-form.png)

### 3.3 查看和修改模型

打开模型后，可以在详情页修改供应商、模型类型、模型名称、认证信息和模型参数。

1. 在列表中点击模型卡片。
2. 或点击卡片右上角菜单后进入详情。
3. 修改内容后点击右上角 **保存**。

![打开模型](/img/manual/model/model-open.png)
![模型详情](/img/manual/model/model-details.png)

### 3.4 删除模型

1. 将鼠标移到模型卡片上。
2. 点击右上角菜单按钮。
3. 选择 **删除**。
4. 在确认框中再次确认。
![删除模型](/img/manual/model/model-show-delete.png)
![删除模型](/img/manual/model/model-delete.png)

### 3.5 维护模型参数

模型参数用于补充模型的可调参数。

1. 在新建模型弹窗或模型详情页中找到 **模型参数**。
2. 点击 **添加参数**。
3. 在右侧抽屉中填写参数内容。
4. 保存后返回参数列表。
![模型参数新增](/img/manual/model/model-params-add.png)
![模型参数抽屉](/img/manual/model/model-params-drawer.png)
