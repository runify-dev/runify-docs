# 项目

## 1 功能说明

项目模块用于统一管理项目目录和项目。你可以在这里创建项目、维护文件夹，并进入项目下的处理器页面。

---

## 2 页面说明

项目页面由两部分组成：

| 区域 | 说明 |
| --- | --- |
| 左侧目录树 | 管理文件夹和项目 |
| 右侧内容区 | 查看项目列表和项目内部页面 |

### 2.1 列表页

列表页中主要包含：

- 搜索框
- 新建项目按钮
- 项目卡片
- 卡片操作菜单

![项目页面](/img/manual/project/project-home.png)

---

## 3 操作步骤

### 3.1 新建项目

1. 进入 **项目** 页面。
2. 在左侧选择目标文件夹。
3. 点击右上角 **新建项目**。

![新建项目按钮](/img/manual/project/project-create-button.png)

也可以从左侧目录树中创建：

1. 在左侧找到目标文件夹。
2. 点击节点右侧操作菜单。
3. 选择 **创建 -> 项目**。

![目录树菜单](/img/manual/project/project-tree-menu.png)

4. 上传项目图标。
5. 填写项目名称。
6. 填写项目描述。
7. 填写项目路径前缀。
8. 点击 **创建**。

![新建项目弹窗](/img/manual/project/project-create-dialog.png)

### 3.2 打开项目

1. 在列表中点击项目卡片。
2. 或点击项目卡片右上角菜单后选择 **打开**。
3. 打开后，左侧菜单会切换为项目内部菜单。
4. 默认进入 **处理器** 页面。

![打开项目](/img/manual/project/project-open.png)
![项目菜单](/img/manual/project/project-menu.png)

### 3.3 查看项目处理器

进入项目后，可以在处理器页面继续查看和维护项目相关内容。

1. 打开目标项目。
2. 在左侧菜单中点击 **处理器**。
3. 在右侧查看项目处理器内容。

![项目处理器](/img/manual/project/project-processor.png)

### 3.4 删除项目

1. 将鼠标移到项目卡片上。
2. 点击右上角菜单按钮。
3. 选择 **删除**。
4. 在确认框中再次确认。

![删除项目](/img/manual/project/project-delete.png)
