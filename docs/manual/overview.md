# 功能手册总览

功能手册用于帮助你按页面完成实际操作。你可以把它理解成一组面向系统使用者的操作说明。

如果你还没有进入系统，建议先阅读 [快速开始](../getting-started/quick-start)。  
如果你已经进入系统，想按页面学习功能，可以从这里开始。

## 阅读说明

功能手册中的页面，通常会包含这些内容：

- 功能说明
- 页面说明
- 操作步骤
- 截图示意

## 当前手册内容

| 模块 | 说明 |
| --- | --- |
| [应用](./application-management) | 创建应用、查看概览、维护工作流、查看对话日志 |
| [模型](./model-management) | 新建模型、维护模型配置和参数 |
| [知识库](./knowledge-management) | 创建知识库、维护文档和知识库设置 |
| [技能](./skill-management) | 创建技能、维护技能文件、配置技能参数、使用技能商店 |
| [数据源](./datasource-management) | 新建数据源、查看详情、维护表和字段信息 |
| [项目](./project-management) | 创建项目、进入项目、查看项目处理器 |

## 建议阅读顺序

1. 先看 [应用](./application-management)
2. 再看 [模型](./model-management)
3. 然后按需要阅读 [知识库](./knowledge-management)、[技能](./skill-management)、[数据源](./datasource-management)
4. 最后看 [项目](./project-management)
