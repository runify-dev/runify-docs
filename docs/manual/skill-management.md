# 技能

## 1 功能说明

技能模块用于统一管理技能、技能文件和技能参数。你可以在这里创建技能、导入技能、维护技能文件，并从技能商店安装或升级技能。

---

## 2 页面说明

技能页面由两部分组成：

| 区域 | 说明 |
| --- | --- |
| 左侧目录树 | 管理文件夹、技能和技能文件树 |
| 右侧内容区 | 查看技能列表、技能设置、文件内容和技能商店 |

### 2.1 列表页

列表页中主要包含：

- 搜索框
- 技能商店按钮
- 创建按钮
- 技能卡片
- 卡片操作菜单

![技能页面](/img/manual/skill/skill-home.png)

### 2.2 技能商店

技能商店页面中主要包含：

- 搜索框
- 技能卡片
- 技能详情弹窗
- 版本历史
- 安装和升级按钮

![技能商店](/img/manual/skill/skill-open-store.png)
![技能商店](/img/manual/skill/skill-store.png)

## 3 操作步骤

### 3.1 新建技能

1. 进入 **技能** 页面。
2. 在左侧选择目标文件夹。
3. 点击右上角 **创建**。
4. 选择 **创建技能**。

![创建技能按钮](/img/manual/skill/skill-create-button.png)

也可以从左侧目录树中创建：

1. 在左侧找到目标文件夹。
2. 点击节点右侧操作菜单。
3. 选择 **创建 -> 技能**。

![目录树菜单](/img/manual/skill/skill-tree-menu.png)

4. 上传技能图标。
5. 填写技能名称。
6. 填写技能描述。
7. 点击 **创建**。

![新建技能弹窗](/img/manual/skill/skill-create-dialog.png)

### 3.2 导入和导出技能

1. 在技能列表页点击 **创建**。
2. 选择 **导入技能**。
3. 选择本地 ZIP 文件后完成导入。
4. 如需导出技能，将鼠标移到技能卡片上。
5. 点击右上角菜单按钮。
6. 选择 **导出**。

![导入技能](/img/manual/skill/skill-import.png)
![导出技能](/img/manual/skill/skill-export.png)

### 3.3 打开技能和查看文件

1. 在列表中点击技能卡片。
2. 打开后，左侧会切换为技能文件树。
3. 在左侧可以查看文件夹、文本文件和其他文件。
4. 点击左侧底部 **设置** 可以进入技能设置页。

![技能文件树](/img/manual/skill/skill-file-tree.png)

### 3.4 管理技能文件

进入技能后，可以在文件树中继续维护文件内容。

1. 打开目标技能。
2. 在左侧文件区点击 **新建文件夹**、**新建文本** 或 **上传文件**。
3. 在文件树中选择目标文件或文件夹。
4. 使用节点菜单进行重命名、删除或继续查看。

![技能文件操作](/img/manual/skill/skill-file-actions.png)
![技能详情](/img/manual/skill/skill-details.png)

### 3.5 编辑技能文件内容

1. 在文件树中点击目标文本文件。
2. 系统会根据文件类型打开对应编辑器。
3. 修改内容后按页面提示保存。

![文件编辑](/img/manual/skill/skill-editor.png)

### 3.6 设置技能参数

1. 打开目标技能。
2. 点击左侧底部 **设置**。
3. 在设置页中修改技能名称和描述。
4. 在 **参数定义** 中添加、编辑或删除参数。
5. 在 **参数值** 中填写具体参数内容。
6. 点击右上角 **保存**。

![技能设置](/img/manual/skill/skill-setting.png)

### 3.7 使用技能商店

1. 在技能列表页点击 **技能商店**。
2. 在商店中搜索目标技能。
3. 点击技能卡片查看详情、能力、依赖和版本历史。
4. 点击 **安装** 安装指定版本。
5. 如果技能已有新版本，可以点击 **升级**。

![技能商店](/img/manual/skill/skill-store.png)
![技能商店详情](/img/manual/skill/skill-store-detail.png)

### 3.8 删除技能

1. 将鼠标移到技能卡片上。
2. 点击右上角菜单按钮。
3. 选择 **删除**。
4. 在确认框中再次确认。

![删除技能](/img/manual/skill/skill-delete.png)
