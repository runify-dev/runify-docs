# Runify

**轻量级白盒 AI Agent 平台**

低资源占用 · 可视化构建 · 完全可控的自定义 AI Agent

> 💡 **告别黑盒与魔法，一切尽在掌握。** 在 Runify 中，你看到的每一步推理、每一次工具调用、每一个决策节点，都是**完全透明、可编辑且可调试**的。我们致力于将算法与工程的确定性，重新完整地交还给开发者。

## 它解决什么问题

大多数 Agent 框架把推理过程藏在黑盒里，调试困难、行为不可控。Runify 把每一个决策节点都摊开在一张可视化画布上，让你像搭积木一样组合 Agent，并且随时介入、编辑、调试。

## 核心特性一览

| 特性 | 说明 |
| --- | --- |
| **白盒可控** | 推理、工具调用、决策节点全程透明可编辑可调试 |
| **轻量运行** | 内存占用 100MB~200MB，启动 1~2 秒，1 核 2G 即可跑满本地流式响应 |
| **可视化构建** | 拖拽式 DAG 画布，摒弃复杂 BPMN，符合开发者直觉 |
| **智能笔记** | 内置富文本编辑器，作为 Agent 的上下文来源与私人知识库 |
| **多数据源** | 一键连接 PostgreSQL / MySQL / Redis 等外部业务环境 |
| **多端适配** | Web 响应式 + Electron 桌面端（macOS / Windows） |

## 适合谁

- 想要**完全掌控** Agent 行为、不接受黑盒的开发者
- 资源有限、希望在**本地或入门级服务器**上跑 Agent 的团队
- 需要把 Agent 接入**私有业务数据**的场景
